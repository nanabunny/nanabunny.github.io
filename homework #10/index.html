Charts built-in

Now I’m trying to visualize my dataset with the Chart built-in.

Based on Idyll’s ‘built-in’ components, I tried to draw a scatter plot, of which x represents speed of pokemon, y represents the total stats of pokemon, indicating how strong it is. Although the graph is rough and simple, we can still see that there is no obvious relationship between the speed and how strong a pokemon is.

Use vega-lite to plot my data

In the plot above, x represents the HP, and y represents the total stats of each pokemon, which means how strong a pokemon is. And we can see that this plot is more meaningful than the previous one, because there is a relationship that total grows with the HP. That is to say, in general, pokemons with higher HP values will be stronger.
